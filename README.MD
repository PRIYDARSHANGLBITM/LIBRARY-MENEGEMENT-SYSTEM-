# 📚 Library Management System

A modern and responsive **Library Management System** developed using **Django**, **Python**, **Bootstrap 5**, **HTML**, **CSS**, and **SQLite**. This project helps manage books, students, book issuing, and library records through separate Admin and Student portals.

---

# ✨ Features

- 🔐 Admin Login & Authentication
- 👨‍🎓 Student Login & Authentication
- 📖 Add New Books
- 📚 View Available Books
- 🗑️ Delete Books
- 👥 Register Students
- 📋 View Students
- 🤝 Issue Books
- 📑 View Issued Books
- 📅 Due Date & Fine Calculation
- 📊 Admin Dashboard
- 🎓 Student Dashboard
- 📱 Fully Responsive UI
- 🎨 Bootstrap 5 + Font Awesome
- 🔒 Role-Based Authentication

# 🚀 Installation

## Clone Repository

```bash
git clone https://github.com/PRIYDARSHANGLBITM/library-management.git
```

Move to Project

```bash
cd library-management
```

Create Virtual Environment

```bash
python -m venv venv
```

Activate Virtual Environment

### Windows

```bash
venv\Scripts\activate
```

### Linux / Mac

```bash
source venv/bin/activate
```

Install Dependencies

```bash
pip install -r requirements.txt
```

Apply Migrations

```bash
python manage.py migrate
```

Run Server

```bash
python manage.py runserver
```

Open Browser

```
http://127.0.0.1:8000
```
